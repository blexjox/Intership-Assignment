# Web Portfolio (Vite + React)

Minimal developer portfolio scaffold using Vite + React.

## Setup

Install dependencies and run the dev server:

```bash
cd web-portfolio
npm install
npm run dev
```

Open http://localhost:5173 (Vite will open automatically).

## Structure

- `index.html` – app entry
- `src/main.jsx` – React bootstrap
- `src/App.jsx` – top-level router and layout
- `src/components/*` – UI components
- `src/styles.css` – global styles

You can customize names, links and add your projects in `src/components/Projects.jsx`.
